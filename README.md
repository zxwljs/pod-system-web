# 叮当跨境ERP · POD 智能套图系统

面向跨境电商卖家（Temu / Amazon / Shopify 等）的一站式 POD（按需打印，Print on Demand）套图工具。把设计稿快速套到各类商品模板上，自动产出合规的上品图，省去大量手工修图的重复劳动。

## 核心功能

### 批量智能套图
- 一次导入多个 SKU，批量完成「设计稿 → 商品图」的套版，不用逐个手动处理。
- 主图与细节图分开处理：主图用于商品首图；细节图可分别输出 **plain（纯图）** 与 **mockup（场景图）** 两种版本，适配不同平台的上品要求。

### 精准定位
- 在模板上以**像素级精度**摆放设计，坐标按原始图片像素记录，套图位置稳定一致，换 SKU 也不跑偏。

### 版权与侵权检测
- 上品前自动做版权 / 侵权风险检测，降低平台下架与投诉风险。
- **安全区保护**：检测认定的安全区域不会被自动修改，避免误伤合规内容。

### 实时任务进度
- 套图任务实时显示进度与状态，大批量任务也能随时看清跑到哪一步、是否出错。

### 批量打包下载
- 任务完成后一键打包下载（ZIP），主图支持**双路径输出**，直接对接你的上品流程。

### 妙手 ERP 对接
- 可配置妙手 ERP 账号，连接后便于把采集箱商品与套图结果打通，减少跨系统搬运。

## 使用方式
1. 下载并启动后端服务,官网：https://ddddnet.cn
2. 在前端连接本地后端，导入设计稿与商品模板。
3. 配置套图规则，提交任务，查看实时进度。
4. 任务完成后批量下载套图结果。

> 测试环境（dev 分支部署）：https://dev.ddddnet.cn —— 用于新功能验证，与生产共用同一套本地后端（`localhost:3001`），数据互通。
